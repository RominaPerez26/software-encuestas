# Proyecto de Aula Programación 2025-2
En el siguiente proyecto, dentro de una carpeta general que contiene todo, llamada "Software-encuentas", podrá encontrar
varias ramas que contienen código y demás funciones para que el mismo funcione.
## Descripción del problema:
El jefe de Talento Humano de una prestigiosa universidad cuenta con una base de datos en un archivo CSV llamado DataBase.csv. Este archivo contiene la información de todos los estudiantes matriculados en el último semestre. Sin embargo, los datos están desorganizados, lo que dificulta la extracción de indicadores clave para una presentación sobre los resultados de la matrícula.

Se encomienda a un grupo de ingenieros el diseño e implementación de un programa en Python que procese esta información de manera eficiente y genere reportes útiles.
